## FRANK MANGA+ v0.9.6

Small reader preference release: page layout is now remembered per manga title.

### Reader layout preferences

- The reader now saves the page layout mode per title: `single`, `double`, or `double-cover`.
- Reopening a title restores that title's last chosen layout, instead of applying the most recent layout from a different manga.
- Titles without a saved layout still fall back to the global/default page mode.
- Changing a title's layout also updates the global fallback for titles that do not have their own saved preference yet.

### Tests + verification

- Added read-state unit coverage for per-title page-mode storage, fallback, and invalid stored values.
- `./verify.sh`: all green — Rust tests/clippy, Vitest, `svelte-check`, production build, and Vite dev CSS probe.

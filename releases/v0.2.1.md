## FRANK MANGA+ v0.2.1

CI infrastructure release. No app-facing changes from v0.2.0.

### What changed

- Per-target Rust build cache so concurrent matrix jobs don't share a slot.
- Bun dependency cache so reruns skip the install round-trip.

Reruns of failed jobs on this tag onward should be markedly faster — the dependency graph compiles once per platform, not once per attempt.
